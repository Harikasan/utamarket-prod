# UTAMarket

AI Powered E-Commerce Website

## Description

UTAMarket is an innovative e-commerce platform that leverages AI recommendations to enhance the shopping experience for the students of UTA with UTA merch. Our goal is to provide users with personalized recommendations and a seamless shopping journey.

## Features

- AI-driven product recommendations
- User-friendly interface
- payment processing
- Order tracking and management
- Responsive design for mobile and desktop
- Live Chat features
- Admin Dashboard

## Installation

To get started with UTAMarket, clone the repository and install the necessary dependencies:

```bash
npm install
```

## Usage

To run the application locally, use the following command:

```bash
npm run dev
```

Visit `http://localhost:3000` in your browser to view the application.
